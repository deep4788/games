Shootin' Goobers
================

A python game written using Pygame where the goal is to shoot the peanuts with a chocolate-firing-cannon.

To Play
-------
```sh
python game.py
```
Use keys **a** and **d** to move left and right and **right shift** to shoot the chocolate drop.
